Qt RGB LED Controller
=============================================

A demonstration of serial communication in Qt to control an RGB LED with an arduino.


Repository Contents
-------------------
* **/Serial_RGB_LED_arduino** - Arduino sketch.
* **/Serial_RGB_Qt** - Qt Project files.
